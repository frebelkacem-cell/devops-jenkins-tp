Contexte general

Environnement de travail

Pour ce cours, j'ai choisi de faire tourner Jenkins dans un conteneur Docker (jenkins/jenkins:lts-jdk21) plutot que de l'installer directement sur ma machine — plus propre, plus facile a reinitialiser en cas de probleme. Jenkins est accessible sur http://localhost:8081.

Pour que les differents services du projet (Jenkins, SonarQube, Tomcat, l'agent de build) puissent communiquer entre eux par leur nom de conteneur, j'ai cree un reseau Docker dedie : devops-net.

Plutot que de changer de projet a chaque TP, j'ai garde une seule application Java/Maven — une petite calculatrice — du debut a la fin. Le meme code est compile, teste, analyse puis deploye au fil des chapitres, avec un pom.xml qui evolue progressivement. C'est cette continuite qui structure toute la suite du rapport.

7

Jenkins Pipeline — Jenkinsfile

Versionner la definition du build

Passage d'une suite de jobs Freestyle a un pipeline declaratif unique, dont la definition vit dans le depot Git (Pipeline script from SCM) plutot que dans l'interface Jenkins.

pipeline {
  agent any
  stages {
    stage('Checkout') { steps { git url: '...devops-jenkins-tp.git', branch: 'main' } }
    stage('Build')    { steps { sh 'mvn clean package' } }
    stage('Test')     { steps { junit 'target/surefire-reports/*.xml' } }
    stage('Summary')  { steps { echo 'Pipeline mis a jour via webhook GitHub.' } }
  }
}

Architecture maitre-esclave

Repartir la charge de build

Ajout d'un agent Jenkins (jenkins/inbound-agent) connecte au controleur via WebSocket, sur le reseau devops-net. Les builds se repartissent ensuite entre le controleur et l'agent selon l'etiquette du job.

Conclusion

Ce que je retiens

Garder le meme projet calculator-app d'un TP a l'autre m'a permis de voir comment toutes ces briques s'assemblent reellement dans un pipeline, plutot que de les traiter comme des exercices isoles. Les quelques erreurs rencontrees en route — bug d'encodage, oubli de push avant une analyse SonarQube, incompatibilite de version Tomcat — ont finalement ete aussi formatrices que les etapes reussies du premier coup.
